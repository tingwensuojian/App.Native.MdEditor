# Docker 数据目录（卷映射）

此目录为 Markdown 编辑器的**文件树根目录**，通过 Docker volume 映射到容器内 `/app/data`。

## 映射关系

| 宿主机路径 | 容器内路径 | 说明 |
|------------|------------|------|
| `./data`   | `/app/data` | 编辑器的可访问文件根，新建/编辑的 Markdown 会持久化到此目录 |

## 使用说明

- 在此目录下创建子目录和 `.md` 文件，即可在编辑器的文件树中看到并编辑
- 所有修改会直接保存到宿主机，容器重启后数据保留
- 可通过 `docker-compose.yml` 修改挂载路径，映射到其他宿主机目录
